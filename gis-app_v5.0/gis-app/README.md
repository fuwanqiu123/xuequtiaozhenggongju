# OpenLayers GIS应用

基于OpenLayers和天地图矢量底图的GIS应用，提供多边形绘制、编辑、导入导出等功能。

## 功能特性

### 1. 地图功能
- 使用天地图矢量底图（带注记）
- 初始定位在湖南省湘潭市
- 支持缩放、平移等基本地图操作

### 2. 绘制功能
- **绘制多边形**：支持绘制单个或多个多边形
- **绘制辅助线**：绘制临时参考线，不干扰多边形
- **绘制辅助多边形**：绘制临时参考多边形，用于辅助参考
- **添加辅助文字**：在地图上添加文字标注
- **添加辅助点**：在地图上添加图标点标记（支持多种图标类型和自定义颜色）
- **清除辅助要素**：一键清除所有辅助要素（线、多边形、文字、点）

### 3. 编辑功能
- **选择多边形**：点击选择要操作的多边形
- **编辑多边形**：进入编辑模式，可拖动顶点调整形状
- **顶点操作**：
  - 添加顶点：在编辑模式下，在多边形边上点击添加新顶点
  - 删除顶点：**点击顶点选中后按Delete键删除**
- **删除多边形**：删除选中的多边形
- **选择辅助要素**：点击选择辅助线/多边形/文字/点
- **删除辅助要素**：删除选中的辅助要素

### 4. 数据管理
- **导出GeoJSON**：将当前选中文件的多边形导出为GeoJSON文件
- **导入GeoJSON**：导入多边形GeoJSON文件到文件列表
- **导出辅助要素**：将所有辅助要素（线/多边形/文字/点）导出为单独的GeoJSON文件
- **导入辅助要素**：导入辅助要素GeoJSON文件，恢复原有样式
- **自动居中**：导入数据后自动调整视图居中显示

### 5. 搜索功能
- **地名检索**：搜索兴趣点、道路等地名
- **结果显示**：在地图上标记搜索结果
- **快速定位**：点击搜索结果快速定位到该位置
- **地址显示**：**鼠标悬停在地图上延迟0.7秒自动显示地址**

### 6. 辅助要素系统
- **独立图层**：辅助要素与多边形完全分离，不加入JSON文件列表
- **类型丰富**：支持辅助线、辅助多边形、辅助文字、辅助点
- **图标多样**：辅助点支持12种图标类型（位置、住宅、学校、医院、商店、餐厅、公园、公交、汽车、旗帜、星星、警告、信息）
- **样式自定义**：辅助点支持自定义颜色和大小
- **批量操作**：支持导入/导出所有辅助要素，保留样式设置
- **显示/隐藏**：一键显示/隐藏所有辅助要素，方便查看底层地图
- **右键菜单**：在顶点上右键显示操作菜单
- **状态显示**：实时显示当前模式、各类要素数量等信息
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- **地图引擎**：OpenLayers 8.2.0
- **底图服务**：天地图矢量地图API
- **UI框架**：jQuery 3.6.0
- **图标库**：Font Awesome 6.4.0
- **样式**：自定义CSS

## 使用方法

### 1. 直接使用
1. 确保网络连接正常
2. 用浏览器打开 `index.html` 文件
3. 应用会自动加载天地图底图

### 2. 功能操作指南

#### 绘制多边形
1. 点击"绘制多边形"按钮
2. 在地图上点击绘制多边形顶点
3. 双击或点击第一个顶点完成绘制

#### 绘制辅助点
1. 点击"添加辅助点"按钮（地图标记图标）
2. 在地图上点击要放置点的位置
3. 在弹出的对话框中选择图标类型（位置、住宅、学校、医院等）
4. 选择图标颜色和大小
5. 点击"确认"完成添加

#### 编辑多边形
1. 点击"选择多边形"或直接点击多边形
2. 点击"编辑多边形"按钮进入编辑模式
3. 拖动顶点调整形状
4. **顶点删除**：点击顶点选中（顶点变为红色），然后按键盘Delete键删除该顶点
5. 再次点击"停止编辑"退出编辑模式

#### 管理辅助要素
1. 点击"选择辅助要素"按钮进入选择模式
2. 点击要操作的辅助要素（线/多边形/文字/点）
3. 按"删除"按钮或键盘Delete键删除选中要素
4. 点击"清除所有辅助要素"可一键清除所有辅助要素
5. 使用"导入辅助要素"和"导出辅助要素"进行数据交换

#### 显示/隐藏辅助要素
1. 点击工具栏中的 👁️ "眼睛"图标按钮
2. 再次点击可切换显示/隐藏状态
3. 隐藏状态下所有辅助要素（线/多边形/文字/点）都会被隐藏
4. 此操作不会删除数据，只是控制可见性

#### 导入/导出数据
- **导出**：点击"导出GeoJSON"下载当前所有多边形
- **导入**：点击"导入GeoJSON"选择本地GeoJSON文件

#### 地名搜索
1. 在搜索框输入地名、道路或兴趣点
2. 点击搜索按钮或按Enter键
3. 查看搜索结果并点击定位

#### 地址显示
- **鼠标悬停**：在地图上移动鼠标，延迟0.7秒后自动显示当前位置的地址
- **自动优化**：系统会自动限制API调用频率，避免频繁请求

## 注意事项

1. **天地图API Key**：应用中已配置天地图API Key，如需更换请修改app.js中的`TIANDITU_KEY`变量
2. **浏览器兼容**：建议使用Chrome、Firefox、Edge等现代浏览器
3. **网络要求**：需要能够访问天地图服务（t0.tianditu.gov.cn）
4. **数据安全**：所有操作在本地浏览器中进行，数据不会上传到服务器
