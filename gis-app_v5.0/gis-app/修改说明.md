# GIS-APP 辅助元素图层修改说明

## 修改概述

根据需求，对GIS-APP工具的辅助元素功能进行了以下修改：

1. **默认创建辅助元素图层** - 工具启动时自动创建一个名为"辅助元素图层"的默认图层
2. **移除新建辅助文件功能** - 删除了"新建辅助元素文件"按钮和相关逻辑
3. **新绘制使用默认图层** - 所有新绘制的辅助元素（线、多边形、文字、点）自动存入默认图层
4. **保留导入编辑导出功能** - 导入的辅助元素文件可在列表中选择、编辑、导出

---

## 详细修改内容

### 1. JavaScript 修改 (app.js)

#### 1.1 修改默认图层创建函数
```javascript
// 原函数：createDefaultAssistFile(fileName) - 支持创建多个文件
// 新函数：createDefaultAssistLayer() - 只创建一个默认图层
```
- 函数名改为 `createDefaultAssistLayer()`
- 固定图层名称为 `"辅助元素图层"`
- 固定ID为 `'assist_layer_default'`
- 添加 `isDefault: true` 标记
- 防止重复创建默认图层

#### 1.2 修改绘制函数
以下绘制函数修改为使用默认图层（如果不存在则自动创建）：

- `startDrawLine()` - 绘制辅助线
- `startDrawAssistPolygon()` - 绘制辅助多边形
- `startDrawAssistText()` - 添加辅助文字
- `startDrawAssistPoint()` - 添加辅助点

修改逻辑：
```javascript
// 原逻辑：如果没有选中的辅助元素文件，提示用户先选择
if (!AppState.selectedAssistFile) {
    showMessage('请先选择一个辅助元素文件', 'warning');
    return;
}

// 新逻辑：如果没有选中的辅助元素文件，使用默认图层
if (!AppState.selectedAssistFile) {
    createDefaultAssistLayer();
}
```

#### 1.3 修改清除辅助要素功能
```javascript
// 原逻辑：清除所有文件和要素
AppState.assistFiles = [];
AppState.assistFeatureMap.clear();

// 新逻辑：只清除要素，保留默认图层，移除导入的文件
// 清除默认图层的要素映射，但保留默认图层本身
const defaultLayer = AppState.assistFiles.find(f => f.isDefault);
if (defaultLayer) {
    AppState.assistFeatureMap.set(defaultLayer.id, []);
    defaultLayer.featureCount = 0;
}
// 只清除非默认的辅助元素文件（导入的文件）
AppState.assistFiles = AppState.assistFiles.filter(f => f.isDefault);
```

#### 1.4 修改应用初始化
```javascript
// 原代码：
createDefaultAssistFile(generateDefaultAssistFileName());

// 新代码：
createDefaultAssistLayer();
```

#### 1.5 移除事件绑定
删除了以下事件绑定：
```javascript
// 已删除：
$('#newAssistFile').on('click', function() {
    createDefaultAssistFile(generateDefaultAssistFileName());
});
```

### 3. 功能保留说明

以下功能完全保留：

1. **导入辅助元素文件** - 点击"导入"按钮可导入单个或多个辅助元素文件
2. **导入文件夹** - 支持导入整个文件夹中的辅助元素文件
3. **文件列表显示** - 默认图层和导入的文件都显示在列表中
4. **文件选择** - 点击列表项可选择某个文件
5. **文件定位** - 点击"定位"按钮可缩放到该文件的范围
6. **文件显示/隐藏** - 点击眼睛图标可切换文件可见性
7. **文件导出** - 点击"导出"按钮可导出单个文件
8. **文件移除** - 点击"移除"按钮可删除文件（默认图层不能被移除）
9. **搜索功能** - 支持按文件名搜索辅助元素文件

## 使用说明

### 新用户首次使用
1. 打开工具后，会自动创建：
   - 一个默认学区文件（如：`新建学区_2024-01-01T00-00-00.geojson`）
   - 一个"辅助元素图层"
2. 直接开始绘制辅助元素（线、多边形、文字、点），会自动存入"辅助元素图层"

### 导入外部文件
1. 点击辅助元素文件列表区域的"导入"或"文件夹"按钮
2. 选择要导入的 `.geojson` 文件
3. 导入的文件会显示在列表中，位于"辅助元素图层"下方
4. 点击某个导入的文件可选择它
5. 新绘制的辅助元素会继续存入当前选中的文件（默认是"辅助元素图层"）

### 文件操作
- **选择文件**：点击列表项
- **定位文件**：点击"定位"按钮
- **显示/隐藏**：点击眼睛图标
- **导出文件**：点击"导出"按钮
- **移除文件**：点击"移除"按钮（仅对导入的文件有效）

## 数据结构变化

### 默认图层数据结构
```javascript
{
    id: 'assist_layer_default',
    name: '辅助元素图层',
    importTime: '2024/1/1 00:00:00',
    featureCount: 0,
    isDefault: true  // 标记为默认图层
}
```

### 导入文件数据结构
```javascript
{
    id: 'assist_file_1234567890_1',
    name: '导入的文件名.geojson',
    importTime: '2024/1/1 00:00:00',
    featureCount: 10
    // 注意：没有 isDefault 属性或 isDefault: false
}
```

---

## 注意事项

1. **默认图层不可删除** - "辅助元素图层"不能被移除，只能清空其中的要素
2. **导入文件可删除** - 导入的辅助元素文件可以单独移除
3. **清除功能** - "清除所有辅助要素"会清空所有要素，但保留默认图层
4. **绘制目标** - 新绘制的辅助元素总是存入当前选中的文件（默认是"辅助元素图层"）
